Enhancing the User Experience of 3D Displayed VR Scenes (Visual Saliency)

This research project explores visual saliency for immersive VR environments. The end goal is to optimise visual fidelity and performance based on predicted user attention, using both classical and learning-based saliency estimation methods.

Project Overview

    • Investigated real-time visual saliency in immersive VR to optimise rendering workloads based on eye-tracker data.
    • Integrated both static (image-based) and dynamic (eye-tracking informed) saliency heatmaps into a Unity-based VR application.
    • Benchmarked performance and perceptual quality metrics across multiple experiments on HTC Vive HMD, measuring FPS and subjective scene coherence.
    • Conducted user studies and visual comparisons using captured saliency overlays and heatmap evaluations on gaze data obtained through Tobii Eye Tracker.
    • Documented experimental findings and architectural design in an academic-style paper to support reproducibility and future extensions.

Rendering Pipeline for High-Fidelity Lunar Terrain

I'm currently working on this for my master's project. The goal is to create a high-fidelity Moon terrain rendering for VR environments, using the Vulkan Graphics API. The project is in its early stages.

Project Overview

At this early stage, I am exploring which graphics API best suits the project's requirements.
